throttle Operator

Description: Emits a value from the source Observable, then ignores subsequent source values for a duration determined by another Observable, then repeats this process.


Syntax: throttle(durationSelector: (value: T) => ObservableInput, config: ThrottleConfig = defaultThrottleConfig)

Snippet: 

               {{snippet}}

Use Case: This operator is mostly used for events that can be triggered tens or even hundreds of times per second. The most common examples are DOM events such as scrolling, resizing, mouse movements, and keypress.
